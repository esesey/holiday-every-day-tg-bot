Этот проект я задумал для того, чтобы у меня всегда были свежие идеи для поздравления моей девушки.

Сейчас я планирую структуру проекта.
Сразу задумываю бот неперсонализированным, но с возможностью создавать "профили" с интересами, описанием характера и т.д. тех, кого хочет поздравить пользователь.
Механизм работы будет следующим:

1. Бот получает по api список праздников на текущий день, учитывая локацию пользователя и дату.
2. Пользователь может запросить просто список праздников, попросить придумать поздравление, либо может создать профиль человека для поздравления.
3. Созданные профили закрепляются за аккаунтом пользователя и их можно будет использовать в будущем.
4. Для генерации поздравления будет использоваться api искусственного интеллекта, с подготовленным промптом.

Исходя из этого, бот должен иметь возможность взаимодействия с тремя сущностями: Праздник, Профиль, Поздравление.
Методы для праздника: получить
Методы для профиля: CRUD
Методы для поздравления: получить

На текущий момент реализации надо решить первые задачи:

1. Поиск апи для получения праздников по дате
2. Поиск апи ии, желательно бесплатного и доступного без впн
3. Написание основной сущности бота
4. Описание типов сущностей
5. Добавление сущности бота нужных методов
